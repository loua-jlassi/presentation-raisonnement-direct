# Guide d'Export vers PowerPoint

## Méthode 1 : Export PDF puis Import dans PowerPoint (RECOMMANDÉ)

### Étape 1 : Exporter en PDF depuis Reveal.js

1. Ouvrez `presentation_revealjs.html` dans Chrome
2. Appuyez sur **Ctrl+P** (Windows) ou **Cmd+P** (Mac)
3. Dans la fenêtre d'impression :
   - **Destination** : Enregistrer au format PDF
   - **Mise en page** : Paysage
   - **Marges** : Aucune
   - **Options** : ✅ Cocher "Graphiques d'arrière-plan"
4. Cliquez sur **Enregistrer** et choisissez un emplacement

### Étape 2 : Importer le PDF dans PowerPoint

**Option A - Import direct :**
1. Ouvrez PowerPoint
2. Allez dans **Fichier > Ouvrir**
3. Sélectionnez le PDF que vous venez de créer
4. PowerPoint convertira automatiquement chaque page en slide

**Option B - Insertion comme images :**
1. Ouvrez PowerPoint
2. Allez dans **Insertion > Images > Ce périphérique**
3. Sélectionnez le PDF
4. Chaque page du PDF sera importée comme une image/slide

## Méthode 2 : Utiliser un service en ligne

1. Allez sur https://www.zamzar.com/convert/html-to-pptx/
2. Téléchargez votre fichier `presentation_revealjs.html`
3. Sélectionnez le format de sortie : PPTX
4. Entrez votre email
5. Téléchargez le fichier PPTX généré

---

## Méthode 3 : Utiliser Chrome avec extension

1. Installez l'extension "Save as PDF" dans Chrome
2. Ouvrez la présentation Reveal.js
3. Utilisez l'extension pour sauvegarder en PDF
4. Importez le PDF dans PowerPoint

---

## Astuce : Mode Présentation pour meilleur export

Dans la présentation Reveal.js :
- Appuyez sur **F** pour le mode plein écran
- Appuyez sur **Esc** pour revenir
- Utilisez les flèches pour naviguer
- Appuyez sur **Ctrl+P** / **Cmd+P** pour exporter

---

## Problèmes courants

**Le PDF ne s'ouvre pas dans PowerPoint ?**
- Vérifiez que vous avez sélectionné "Graphiques d'arrière-plan" lors de l'export
- Essayez d'ouvrir le PDF dans un lecteur PDF d'abord pour vérifier qu'il est valide

**Les couleurs ne s'affichent pas correctement ?**
- Assurez-vous d'avoir coché "Graphiques d'arrière-plan" dans les options d'impression
- Utilisez Chrome plutôt qu'un autre navigateur

**Les slides sont trop grandes/petites ?**
- Dans PowerPoint, allez dans **Conception > Taille de la diapositive**
- Ajustez la taille selon vos besoins
